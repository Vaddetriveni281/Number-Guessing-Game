# PythonProjects

This project contains Python-based mini-games implemented in Jupyter Notebooks.

## Projects Included
- Number Guessing Game
- Calculator
- Dice Roller

## How to Run
1. Clone this repository.
2. Open the `.ipynb` file in Jupyter Notebook or VS Code.
3. Run the cells.

## Requirements
- Python 3.x
- Jupyter Notebook
